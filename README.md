 React Tailwind Landing Page

A clean and modern landing page built with React and Tailwind CSS.

## Features

- Responsive header with mobile navigation
- Premium hero section with typed text
- Analytics showcase section
- Pricing-style cards
- Newsletter section
- Custom footer with LinkedIn link

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Icons
- React Typed

## Getting Started

```bash
npm install
npm run dev
```

Frontend landing page project focused on layout, responsiveness, and clean UI structure.
